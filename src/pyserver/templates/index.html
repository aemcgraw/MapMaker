MapMaker

Options

  • Width
    Map Width in Pixels
  • Height
    Map Height in Pixels
  • Chaos
    Approximately represents the macro-level topographical variation of the map.
    Typical values: 0 - 1
    Default value: 0.5
  • Damping
    Approximately represents the micro-level topographical variation of the map.
    Typical values: 0 - 1
    Default value: 0.8
  • Blocksize
    The width/height in pixels of a region of the map.
    This should be a value less than the minimum of the width and height of the map.
  • Seed
    Integer value used to seed the map generation.
    A map can be refined by changing other settings and leaving the seed fixed.
    If this field is blank or set to 0 a random seed will be used instead.
  • Water
    Water level of the map as a percentage.
    e.g a value of 70 will result in a map that is 70% water
    0.5
  • Dark
    Darkness
    1.0
